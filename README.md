# Oil Well Data Preprocessing

## Overview
This project performs exploratory data analysis (EDA) and data preprocessing 
on oil well operation parameters collected between 2013 and 2021.

The goal is to clean and prepare the dataset for downstream machine learning tasks 
such as flow rate prediction or anomaly detection.

## Dataset
- **Source:** [Kaggle — Oil Well by Ruslan Zalevskikh](https://www.kaggle.com/datasets/ruslanzalevskikh/oil-well)
- **Format:** Excel (.xlsx)
- **Period:** 2013–2021
